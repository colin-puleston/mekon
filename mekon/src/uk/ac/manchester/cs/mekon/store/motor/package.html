Provides the classes and interfaces of the MEKON Instance-Store Mechanism API (see {@link uk.ac.manchester.cs.mekon} for general description).

NOTE: This is a companion package to the {@link uk.ac.manchester.cs.mekon.store} package, with which it shares circular dependencies at the implementation level, though not at the API level. See {@link uk.ac.manchester.cs.mekon} for further details.
