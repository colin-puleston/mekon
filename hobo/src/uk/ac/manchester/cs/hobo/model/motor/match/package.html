Provides mechanisms for the domain-specific customisation of the matching mechanisms provided by specific implementations of {@link uk.ac.manchester.cs.mekon.model.motor.IMatcher}, based on particular HOBO Object Models (OM). The {@link uk.ac.manchester.cs.hobo.model.motor.match.DMatcherCustomiser} class is an abstract implementation of {@link uk.ac.manchester.cs.mekon.model.motor.IMatcher} that provides a domain-customised wrapper round a generic "core-matcher", which can be The core-matcher can be provided by any implementation of {@link uk.ac.manchester.cs.mekon.model.motor.IMatcher}.
