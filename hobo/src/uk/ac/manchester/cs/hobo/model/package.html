Provides the classes and interfaces of the HOBO Direct Model API (see {@link uk.ac.manchester.cs.hobo} for general description).

NOTE: This is a companion package to both the {@link uk.ac.manchester.cs.hobo.modeller} and {@link uk.ac.manchester.cs.hobo.mechanism} packages, with each of which which it shares circular dependencies at the implementation level, though not at the API level. See {@link uk.ac.manchester.cs.hobo} for further details.
