Provides the classes and interfaces of the MEKON Frames Model API (see {@link uk.ac.manchester.cs.mekon} for general description). The main point-of-access for the client is {@link uk.ac.manchester.cs.mekon.model.CModel}, which provides methods for accessing and instantiating the entities in the model.

NOTE: This is a companion package to the {@link uk.ac.manchester.cs.mekon.mechanism} package, with which it shares circular dependencies.
