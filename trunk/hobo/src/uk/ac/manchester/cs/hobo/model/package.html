Provides the classes and interfaces of the MEKON Direct Model API (see MEKON API overview for general description). The main point-of-access for the client is {@link uk.ac.manchester.cs.hobo.model.DModel}, which provides methods for accessing and instantiating the entities in the model.

NOTE: This is a companion package to the {@link uk.ac.manchester.cs.hobo.mechanism} package, with which it shares circular dependencies.
